![popup](src/assets/popup.png)

## Installation

```
yarn
```

## Cleaning

```
yarn clean:dist
```

This will delete the `dist` folder.


## Development mode

```
yarn start
```

This mode will work in both Chrome and Firefox.


## Build mode

```
yarn build
```

Warning: This builds into `dist/webext-prod`!
